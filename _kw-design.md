### 除外KW（既存LPに残す or 共食いNG）

| 除外KW | 残すLP | 理由 |
|---|---|---|
| 金 買取 / プラチナ 買取 / 銀地金 買取 | 貴金属LP | 棲み分け方針：貴金属LPは「金・プラチナ・銀地金」専門に縮小予定 |
| K18 / 18金 / 24金 / 999銀 / 純銀 | 貴金属LP | 同上 |
| 楽器 買取 / ギター 買取 / ピアノ 買取 / 管楽器 買取 / 弦楽器 買取 | 楽器LP | 既存楽器LPで完結 |
| 遺品整理 / 生前整理 / 形見分け | 遺品整理LP | 既存遺品LPで完結（ただしブランドLPからの内部リンク導線は残す） |
| ゴミ屋敷 / 不用品処分 / 片付け | ゴミ屋敷LP | 既存ゴミ屋敷LPで完結 |
| 質屋 / 質入れ / 質料 / 利息 | 範囲外 | 自社業態に質預かりなし。kanteikyokuのみ実装 |
| ノーブランド ジュエリー / 切手 / コイン | 検討（雑多買取） | 主要LPからは除外、汎用ページのみ |

**共食い回避の方針**:
- ブランドLP h1/title では「ブランド」を必ず先頭に置き、「金」「楽器」「遺品」を含めない
- ヘッダー/フッター内部リンクで他LPへ送客するが、各LPの主管KWはそれぞれ専用LPで取る
- 「ブランド遺品買取」「ブランド遺品整理」のような複合KWはブランドLPで取り、遺品整理LPからもh3/h4で内部リンクを張る
