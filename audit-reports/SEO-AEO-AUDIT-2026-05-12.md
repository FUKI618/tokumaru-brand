## 3. 改善提案（優先順）

### 🔴 P1 必須（納品基準9.1到達のため、AEO -1.2 のうち1.0を取り戻す）

#### P1-1. dateModified の可視テキスト表示
**作業**: フッターまたは記事末尾に「最終更新日: 2026年5月12日」を表示
**実装**:
```astro
<!-- Footer or after main content -->
<p class="text-sm text-gray-500">最終更新日: 2026年5月12日</p>
```
**期待効果**: Perplexity 鮮度 3.2x lift / Google AI Overviews 12-18ヶ月鮮度シグナル達成

#### P1-2. Person sameAs を 2名分追加
**作業**: 査定士2名 (小山/加藤) に外部権威リンクを `sameAs` で付与
**実装** (Layout.astro の Person schema 内):
```json
"sameAs": [
  "https://www.linkedin.com/in/koyama-tokumaru",
  "https://example.tokumarushokai.com/staff/koyama"
]
```
※LinkedInが無い場合は、専門団体ページ・自社スタッフページ・取材記事のURLでOK
**期待効果**: E-E-A-T citation +3.2x

#### P1-3. メインKW「ブランド買取」を +3回追加 (12→15)
**作業**: 「ブランド買取の流れ」「ブランド買取の実績」など、文脈自然な箇所で3回追加
**実装**: コピー上の `voices.ts` / `faq.ts` / hero副文に1箇所ずつ
**期待効果**: SEO 9.0 → 9.3

### 🟡 P2 推奨（さらに 9.5+ へ）

#### P2-1. Answer-first 弱点セクション (4箇所) の冒頭リライト
- H2 #1, #2, #7 の冒頭40-60語を「結論先出し」型に書き換え
- 例: 「保証書がなくても買取可能。エルメス・シャネル等の主要ブランドはケアキット程度の使用感でも当社の買取対象です。査定料・出張費すべて無料。」

#### P2-2. 動画埋め込み 1本
- 「査定の流れ」または「お客様の声」動画を YouTube に上げて埋め込み
- マルチモーダル +156% selection lift（実証）

#### P2-3. AggregateRating 実装
- 実顧客レビューが集まり次第（Google Maps レビュー5件以上推奨）
- LocalBusiness schema に `aggregateRating` 追加

### 🟢 P3 オプション（運用フェーズ強化）

#### P3-1. 月次 dateModified 更新サイクル設定
- pro-lp-ops M月次モードで、最低でも月1回 substantive content update + dateModified 更新

#### P3-2. description を 145-155文字に伸長
- 現125字 → 145字に伸ばして「形見整理・断捨離・相続」等の検索意図キーワードを追加

#### P3-3. 残り alt=空の `<img>` 1個を修正
