## 数千万LPと比べて足りない3つの本質

### 1. 「世界観の作り込み」が足りない（最重要）
Hermès / Chanel / Apple / Aesop / Loewe の公式LPは、**ブランド固有の言語**（タイポ、グレード、写真様式）を持つ。本LPは「Cormorant Garamond + Noto Serif JP + 深ネイビー」という**汎用的なラグジュアリーテンプレート**の組み合わせで、徳丸商会**独自の視覚的ID**が無い。ロゴの "T" マーク、チャンピオンカラー、カスタムイラスト、独自書体（Display用にカスタムモノグラム）等、**他社には絶対に無い視覚資産**が1つも存在しない。

### 2. 「素材の質」が AI 生成の限界
ヒーロー背景・ベルベットトレイ・金庫・査定室はいずれも **AI生成画像のテクスチャ感**が抜けない（被写界深度が不自然、光源が均質、スケール感が出ない）。数千万LPは必ず **実機材で撮影した4K以上のRAW**＋カラーグレーディングを使う。AI画像は背景の「気配」までしか担えず、**主役のプロダクト写真**として使うと一発で安く見える。

### 3. 「動きの言語」が借り物
Magnetic / Tilt / Custom Cursor / Text Split Reveal — これらは Awwwards の常連手法で、**2018-2022 のトレンド集合体**。数千万LPは（例: Locomotive 製の Bonjour Lab、Active Theory 製の Apple AirPods Pro 等）**そのプロジェクト用に発明された動き**を1つは持っている。本LPは「全部入りだが何も新しくない」状態。

## すぐに改善できる5つの提案（実装1日以内）

### A. ヒーロー h1 のジャンプ率を倍に
現状 `clamp(2.4rem, 7.5vw, 6.4rem)` → **`clamp(3.2rem, 11vw, 9.5rem)`** に。日本語キャッチ `2.4rem` は `1.6rem` まで落として、英語と日本語の階層を**3:1**まで開く。今は1.5:1で平坦。

### B. ボルドーをアクセントカラーとして全体に再分配
現在ボルドーは「最低買取保証額」「人気No.1バッジ」のみ。**章番号インデックス（Ⅰ. Ⅱ. Ⅲ.）と買取金額の数字すべて**にボルドーを使い、視線の停留点を作る。今は全部ゴールドで「目が滑る」。

### C. Cormorant を Italic で封印せず、Display Bold + Italic の二段重ね
chapter-num と display-md がどちらも**同じ Cormorant Italic**で書体ヒエラルキーが消えている。display-md は `Cormorant SC`（Small Caps）または `Bold non-italic` に切り替え、Italic は `chapter-num` 専用の「囁き」用途に限定する。

### D. ヒーロー画像の上に「光のアニメーション」を1つ載せる
現状の picture + parallax だけだと「写真が動くだけ」。**SVG で光の粒子（数十個）を slow drift** させるか、CSS `radial-gradient` を `background-position` で 30秒かけてゆっくり回転させると、ヒーローに「呼吸」が生まれる。

### E. モバイル ヒーローの上下マージンを再設計
現在 `py-24` で上下に空きすぎ、display-hero が `clamp` の下限に張り付いて小さい。モバイルだけ `min-h-100svh` のうち**画像を 60vh** で固定し、テキストは下40vh で `text-3xl` を効かせる二段構成へ（既に MEMORY.md にある「モバイルヒーロー2段構成」ルールの完全適用）。

## 実装しないと数千万LPには届かない3つの大きな施策

### 大施策1: WebGL ヒーロー（Three.js / OGL）
**やること**: ヒーロー背景を「動画 + WebGL シェーダー」に置き換え、ブランドバッグ・時計の **3Dモデルが宙に浮いてゆっくり回転**する。マウス位置に応じて視差。
**理由**: Hermès / Cartier / Bulgari の現行サイトは全社 WebGL 採用済み。「2D の写真 + parallax」は2026年時点で**1ランク下**の打ち手。
**所要**: 3Dモデル発注（HERMES バーキン / ROLEX デイトナ / Cartier ラブリング の3点で十分）+ Three.js 実装で 5-7日。

### 大施策2: ブランド独自の動画コンテンツ（15-30秒のループ）
**やること**: 査定員の手元・ルーペ・ベルベットトレイへの設置を**実写4K**で撮影、Magic Bullet で深いティール&ティールゴールドにグレーディング。ヒーローまたは「査定員紹介」セクションでループ再生。
**理由**: 数千万LPの「**最初の3秒で世界観が伝わる**」を実現する唯一の手段は動画。AI画像では絶対に届かない。
**所要**: 撮影1日（カメラマン+査定員）+ 編集2日。費用15-30万円。

### 大施策3: Custom Display Typeface または Variable Font の導入
**やること**: 徳丸商会専用の**カスタム英字書体**を作る、または **Variable Font** を採用してウェイトを連続変化させる "wow" 演出（例: スクロール量に応じてウェイトが 100→900 へモーフ）。
**理由**: 数千万LPは必ず**書体で勝負**している（Aesop = 専用 Suisse, Loewe = 専用 Loewe Serif）。Cormorant Garamond は Google Fonts の汎用品で、これだけで「テンプレ感」が抜けない。最低でも有料書体（Klim, Pangram Pangram, Dinamo 等）への置換、理想はカスタム。
**所要**: 有料書体購入で1日 / Variable Font 演出実装で2-3日 / フルカスタム書体は外注3ヶ月＆数百万円。

## 補足: なぜ「9.88/10」と「6.55/10」のギャップが出るのか

- 届かせるには、上記「大施策3つ」のうち**最低2つ**を実装する必要がある
